Python基本入门和基本数据类型介绍

Philip.Lee

2017-05-17

Python

基础：

print ， 使用,表示不换行，否则会自动换行
多行，和OC一样，可以使用 \ 进行换行连接
字符串： 单引号，多引号和三引号 ',", ''' 或 """. 三引号可以是多行
注释： 单行注释使用#， 多行使用三引号 ''' 或 """
pass： 不执行任何语句，只是使代码有格式

帮助： 可以使用 help（命令） 来获取命令的帮助信息。获取一个模块中的所有函数名字可以使用 dir(模块)

基本类型

数字类型

  1. int （有符号整型） (最大19位)
  2. long（长整型[也可以代表八进制和十六进制])
  3. float（浮点型）
  4. complex（复数）

字符串 所有函数在模块string中

** 获取特定字符串字符**

** 格式化字符串 %

格式化符号

  • %c 格式化字符及其ASCII码
  • %s 格式化字符串
  • %d 格式化整数
  • %u 格式化无符号整型
  • %o 格式化无符号八进制数
  • %x 格式化无符号十六进制数
  • %X 格式化无符号十六进制数（大写）
  • %f 格式化浮点数字，可指定小数点后的精度
  • %e 用科学计数法格式化浮点数
  • %E 作用同%e，用科学计数法格式化浮点数
  • %g %f和%e的简写
  • %G %f 和 %E 的简写
  • %p 用十六进制数格式化变量的地址
  • %% 输出一个单一的’%’

列表 使用[]表示 所有函数包含在 list模块

一个可变的数组，可以增删改查，支持+运算(合并两个列表)

创建二位数组

元组

元组和列表的区别是，元组是不可变的。一旦生成，不能增删改。如果在程序中以列表的形式传递一个对象的集合，它可能在任何地方改变；如果使用元组的话，则不能。
因为括号()既可以表示tuple，又可以表示数学公式中的小括号。 所以，如果元组只有1个元素，就必须加一个逗号，防止被当作括号运算.

如果需要快速创建元组，可以先创建一个列表，再转为元组

时间格式化

python的时间是在模块time中，可以使用dir(time)查看所有有关time的函数。使用的时候，必须先导入时间模块，引入时间模块import time
获取时间戳： time.time() 以秒为单位，float 类型
值得一提的函数：time.localtime() 返回的是一个结构类型的时间

格式化时间：

  • %y 两位数的年份表示（00-99）
  • %Y 四位数的年份表示（000-9999）
  • %m 月份（01-12）
  • %d 月内中的一天（0-31）
  • %H 24小时制小时数（0-23）
  • %I 12小时制小时数（01-12）
  • %M 分钟数（00=59）
  • %S 秒（00-59）
  • %a 本地简化星期名称
  • %A 本地完整星期名称
  • %b 本地简化的月份名称
  • %B 本地完整的月份名称
  • %c 本地相应的日期表示和时间表示
  • %j 年内的一天（001-366）
  • %p 本地A.M.或P.M.的等价符
  • %U 一年中的星期数（00-53）星期天为星期的开始
  • %w 星期（0-6），星期天为星期的开始
  • %W 一年中的星期数（00-53）星期一为星期的开始
  • %x 本地相应的日期表示
  • %X 本地相应的时间表示
  • %Z 当前时区的名称
