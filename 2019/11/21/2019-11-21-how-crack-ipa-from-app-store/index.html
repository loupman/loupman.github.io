ipa砸壳，frida，frida-ios-dump

Philip.Lee

2019-11-21

iOS

最近对App的逆向安全比较兴趣，准备自己试验一下砸壳App。因为直接从App Store下载安装的App是加壳的（我们也可以通过 AppleConfigurator2下载到本地），其实就是经过加密的，这个“砸壳”的过程就是一个解密的过程。未砸壳的App是无法在Class-dump、Hopper或者IDA等工具中使用的。所以我们要将App进行砸壳处理。

一、越狱手机

硬件设备：iPhone5s iOS12.4.3 （如果没有升级到这个版本，可以使用爱思助手升级到这个版本）
需要软件：checkra1n , pip, frida-tools, frida, frida-ios-dump, usbmuxd

在Mac上安装 checkra1n beta 0.9.2， 根据软件指示操作，指示非常简洁明确，而且容易操作。但是是英文（自己谷歌翻译吧）
我的是在 checkra1n 0.9.2 可以越狱成功，版本0.9.1上越狱失败了。手机越狱成功之后，手机上会有checkra1n 软件，否则越狱失败（一直停留在显示越狱界面）。链接WiFi打开checkra1n下载Cydia App。全程不需要开VPN。

二、安装砸壳环境

这里使用“frida-ios-dump” 作为砸壳工具，手机frida的版本最好和电脑的版本一致

1、在手机上安装Frida(Frida-Server)

启动 Cydia 然后通过 Manage -> Sources -> Edit -> Add这个操作步骤把https://build.frida.re这个代码仓库加入进去。然后你就可以在 Cydia 里面找到 Frida 的安装包了，然后你就可以把你的iOS设备插入电脑，并可以开始使用 Frida 了。

我们使用USB进行砸壳，还需要安装最后一个软件 usbmuxd — iproxy，该工具会将USB链接的设备上的端口号映射到电脑上的某一个端口

以上命令就是把当前连接设备的22端口(SSH端口)映射到电脑的2222端口，那么想和设备22端口通信，直接和本地的2222端口通信就可以了。
所以，SSH连接越狱iPhone设备就可以这样连接了
